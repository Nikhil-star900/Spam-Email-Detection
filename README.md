This project uses Machine Learning and NLP to classify emails as Spam or Not Spam (Ham). Email text is cleaned and converted into features using TF-IDF / Bag of Words, and models like Naive Bayes or Logistic Regression are trained to detect spam emails.

 Tech Stack

Python, Pandas, NumPy, Scikit-learn, NLTK

 Goal

Automatically filter unwanted emails and improve inbox security.

 Run
 pip install -r requirements.txt
jupyter notebook
